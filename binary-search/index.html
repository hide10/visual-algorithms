本文

しくみと考え方

整列済み配列に対して高速に探索する方法です。毎回半分の候補を捨てられるため O(log n) になります。

考え方

擬似コード

low = 0, high = n - 1
while low <= high
  mid = floor((low + high) / 2)
  if a[mid] == target return mid
  if a[mid] < target then low = mid + 1 else high = mid - 1
