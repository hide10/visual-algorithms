概要

概要

新しい要素を 1 つ取り出し、左側の整列済み領域の適切な位置へ挿入します。ほぼ整列済みの入力では強いアルゴリズムです。

考え方

擬似コード

for i = 1 to n - 1
  key = a[i]
  while j >= 0 and a[j] > key
    a[j + 1] = a[j]
  a[j + 1] = key

要点まとめ
